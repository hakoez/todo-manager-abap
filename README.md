ABAP To-Do Manager (Cloud-ABAP / SAP BTP Trial)

Ein kleines Übungsprojekt, um die Grundlagen von Cloud-ABAP in der SAP BTP zu lernen.
Das Projekt läuft über die Schnittstelle IF_OO_ADT_CLASSRUN und zeigt, wie man einfache Datenstrukturen und interne Tabellen in ABAP benutzt.

Was dieses Projekt macht

  -Aufgaben hinzufügen (add_task)
  
  -Aufgaben als erledigt markieren (mark_done)
  
  -Aufgaben löschen (delete_task)
  
  -Ausgabe der Aufgabenliste über out->write()
  
  -Alle Daten werden in einer internen Tabelle gespeichert.

Technologien

  -SAP BTP ABAP Environment (Trial)
  
  -Cloud-ABAP
  
  -Eclipse ADT (ABAP Development Tools)

Was ich mit dem Projekt gelernt habe

  -Arbeiten mit Klassen in ABAP
  
  -Erstellen von Strukturen (ty_task)
  
  -Nutzen von internen Tabellen
  
  -Verwenden von:
  
  -LOOP AT … ASSIGNING
  
  -WHERE-Filter
  
  -DELETE … INDEX sy-tabix
  
  -VALUE-Operator
  
  -INSERT
  
  -Starten eines ABAP-Programms über IF_OO_ADT_CLASSRUN

Wie man das Projekt ausführt

  1.Klasse ZCL_TODO_MANAGER öffnen
  
  2.Rechtsklick → Run As → ABAP Application (Console)
  
  3.Die Ausgabe erscheint in der ADT-Konsole

Warum dieses Projekt?

  -Es ist ein einfacher Einstieg, um:
  
  -die ABAP-Sprache zu lernen
  
  -Objektorientierung in ABAP zu verstehen
  
  -interne Tabellen zu benutzen
  
  -später die gleiche Logik als RAP/Fiori-App umzusetzen
